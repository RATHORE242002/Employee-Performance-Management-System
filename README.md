# Employee-Performance-Management-System

Employee Performance Management System 
Technologies Used
Python
SQLite
Pandas
Matplotlib
Seaborn

Developed a Python-based Employee Performance Management System with SQLite database integration for employee record management.
Implemented CRUD operations (Create, Read, Update, Delete) for maintaining employee performance data.
Designed interactive data visualizations using Matplotlib and Seaborn, including:
Employee Performance Bar Chart
Department-wise Average Performance Chart
Top 5 Performers Analysis
Performance Score Distribution Histogram
Department Comparison Box Plot
Utilized SQL queries for efficient data storage, retrieval, and updates.
Implemented exception handling and menu-driven functionality for improved usability and error management.
